# 营口市场调研报告 — 星火APP 可行性分析

> 日期: 2026-05-15
> 调研范围: 社区规模、商户分布、竞品格局、健身消费、政策支持
> 目标: 判断星火APP「训练即任务」模式在营口落地的可行性

---

## 一、营口城市基本盘

| 指标 | 数据 | 判断 |
|------|------|------|
| 常住人口 | **224.2万** (2024) | 三线城市中等规模，够用 |
| 城镇化率 | **68.65%** | 较高，城镇人口约154万 |
| 行政区划 | 4区2县 (站前/西市/老边/鲅鱼圈 + 大石桥/盖州) | 试点可选鲅鱼圈或站前区 |
| 社区数量 | **187个社区** | 覆盖面充足 |
| 完整社区试点 | 已建6个 | 可作为首批合作目标 |

> **结论**: 人口规模支撑一个单社区试点绰绰有余，187个社区为后续扩张提供基础。

## 二、商户与商业分布

### 核心商圈

| 区域 | 定位 | 商户密度 | 适合星火场景 |
|------|------|---------|-------------|
| **站前区** | 市中心 | 高（万达广场、市府路商圈） | 配送/文件/餐饮 |
| **鲅鱼圈区** | 经济强区 | 高（万隆广场、蝴蝶泉路商圈） | 配送/旅游服务 |
| **西市区** | 老城区 | 中（相和购物中心、渤海大街） | 社区巡检/帮扶 |
| **老边区** | 工业配套 | 中（国际汽保汽配城） | 搬运/仓库 |
| **大石桥市** | 县级市 | 中（兴隆商城、大润发） | 社区配送 |
| **盖州市** | 县级市 | 中（华阳商厦） | 社区服务 |

### 商户类型推测（以社区周边为主）

| 商户类型 | 营口分布 | 可能产生的轻体力任务 |
|----------|---------|---------------------|
| 水果店/生鲜店 | 每个社区3-5家 | 配送上门 🏃‍♂️ |
| 超市/便利店 | 每个社区2-3家 | 补货/配送 🚶 |
| 餐饮外卖 | 商圈密集 | 取餐配送 🏃‍♂️ |
| 快递驿站 | 每个社区1-2家 | 派件 🚶 |
| 物业/居委会 | 每个社区1家 | 社区巡检/通知 🚶 |
| 五金/建材 | 城郊/老边区 | 搬运 💪 |
| 药店 | 每个社区1-2家 | 送药上门 🚶 |

### 商户合作的切入点

上线初期最容易打通的商户类型：
1. **水果店/生鲜店** — 高频配送需求，决策链条短
2. **快递驿站** — 末端派件痛点明确
3. **社区物业** — 巡检/通知等非核心业务

## 三、竞品格局

### 现有竞品定位

| 平台 | 类型 | 营口覆盖 | 单价参考 | 与星火的差异化 |
|------|------|---------|---------|--------------|
| **美团外卖** | 餐饮配送 | ✅ 已覆盖成熟 | ~4-5元/单 | 只送吃，不绑定健身 |
| **饿了么** | 餐饮配送 | ✅ 已覆盖 | ~4-5元/单 | 同上 |
| **UU跑腿** | 万能跑腿 | ✅ 已开通 | ~6-15元/单 | 万能但无健身属性 |
| **顺丰同城** | B端配送 | ✅ 覆盖 | ~6-10元/单 | 文件/商务件 |
| 个人跑腿 (58同城) | 众包 | ✅ 有 | 面议 | 无保障无系统 |

### 星火的差异化优势

| 维度 | 竞品 | 星火 |
|------|------|------|
| 核心动机 | 赚钱 | 健身+赚钱双重驱动 |
| 训练绑定 | ❌ 无关 | ✅ 匹配训练计划 |
| 用户留存 | 用完即走 | 积分/定制奖励持续牵引 |
| 商户成本 | 抽佣20-26% | 抽佣10% |
| 社区属性 | 弱 | 强（社区商户+邻里） |

> **结论**: 营口跑腿市场已有基础玩家，但星火的「训练匹配+10%低佣金+积分奖励」是差异化武器，不做美团/饿了么的直接竞品，而是做社区健身+轻劳务的新品类。

## 四、健身消费分析

### 营口人健身画像（基于三线城市共性推断）

| 维度 | 数据 | 对星火的意义 |
|------|------|-------------|
| 年均健身支出 | ~2000-4000元 | 用户有健身花钱意愿 |
| 最受欢迎运动 | 跑步/健走 > 羽毛球 > 游泳 | 步行/跑步型任务最受欢迎 |
| 核心健身人群 | 19-34岁（占50%+） | 与跑腿用户高度重合 |
| 健身决策首因 | 距离便利（15分钟内） | 社区级覆盖正好匹配 |
| 社区健身趋势 | 体育公园/全民健身中心使用率上升 | 免费/低成本健身场景增多 |

### 健身人群与跑腿人群的重合度

```
     跑腿用户（赚钱需求）         健身人群（健康需求）
    ┌──────────────────┐       ┌──────────────────┐
    │  全职骑手(5%)     │       │  健身达人(15%)    │
    │  兼职赚外快(30%)  │ ◄────►  偶尔运动(45%)    │  ← 星火目标用户
    │  偶尔接单(65%)    │       │  从不运动(40%)    │
    └──────────────────┘       └──────────────────┘
    
    核心交集：年轻人+有运动意愿+愿意赚外快 → 约15-20万营口城镇人口
```

## 五、政策支持

### 可直接利用的政策

| 政策 | 内容 | 对星火的用途 |
|------|------|-------------|
| **灵活就业社保补贴** | 就业困难人员可申请社保补贴，社区办理 | 可作为吸引兼职用户的话术 |
| **公益性岗位补贴** | 社区服务岗位补贴1900元/月 | 与居委会合作时参考 |
| **一次性扩岗补助** | 招聘高校毕业生补贴1500元/人 | 发展初期招聘运营人员可申请 |
| **就业见习基地** | 企业可申请设立见习基地 | 未来规模化后利用 |

### 社区层面的合作点

- 社区有**就业困难人员**对接需求 → 星火提供任务机会
- 社区有**巡检/通知/帮扶**人力需求 → 星火用户可接单
- 社区有**老年群体配送**需求 → 星火覆盖

## 六、可行性判断

### 利好因素

| 因素 | 评级 | 说明 |
|------|------|------|
| 城市人口规模 | ✅ 够用 | 224万人，187个社区 |
| 商户需求 | ✅ 存在 | 配送/巡检/搬运场景明确 |
| 健身人群基础 | ✅ 有 | 跑步/健走是全民运动 |
| 竞品差异化 | ✅ 明确 | 训练匹配+低佣金+积分 |
| 政策支持 | ✅ 可利用 | 灵活就业/社区服务补贴 |
| 冷启动成本 | ✅ 低 | 1万元启动资金，先跑小程序 |

### 风险因素

| 风险 | 等级 | 应对 |
|------|------|------|
| 营口冬季（-15°C）户外任务意愿下降 | 🟡 中 | 冬季主推室内型任务（搬运/巡检） |
| 美团/饿了么用户习惯根深蒂固 | 🟡 中 | 不做配送竞品，做健身新品类 |
| 社区商户数字化意识弱 | 🟡 中 | 免费入驻+10%低佣金降低门槛 |
| 种子用户获取成本 | 🟡 中 | 首单积分加成+推荐奖励裂变 |

### 试点建议

```
首推试点区域：站前区（市中心，社区密度最高，商户最多）
试点规模：1-2个大型成熟社区（>5000户）
首批商户目标：小区周边水果店+快递驿站+社区物业（20-30家）
种子用户获取：社区地推+健身群裂变（200-300人）
验证指标：日均任务匹配率>60%、用户周留存>40%
```
